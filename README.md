# Tesla and GameStop Stock and Revenue Dashboard

## Project Overview
This repository contains the final project for the Data Science course. It demonstrates:

- Extracting stock data using yfinance (Tesla and GameStop)
- Extracting revenue data using web scraping (Tesla and GameStop)
- Cleaning and preprocessing data
- Visualizing historical stock prices and quarterly revenues using interactive dashboards

## Files
- `Stock_Analysis_Assignment.ipynb` — Jupyter notebook with all code and outputs  
- `screenshots/` — Folder containing screenshots of the notebook outputs and dashboards  

## How to Run
1. Open the notebook in JupyterLab, Jupyter Notebook, or Google Colab.  
2. Execute the cells step by step to reproduce the stock and revenue analysis.  
3. View the interactive dashboards for Tesla and GameStop.
